git init .                   -> cria um repogit na pasta atual
git status                   -> mostra o status do trabalho 
git commit - m "msg"         -> faz commit do que estava para ser comitado
git add nome_arquivo -> adiciona arquivo para ser comitado
git log -> mostra todo o histórico de commit
git dif -> mostra diferença do estado atual para o estado do ultimo commit


##Fluxo de trabalho

1. Criar o diretório do projeto
2. Iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivo>
6. git status
7. git commit -m "msg"
8. git status
9. volta ao passo 3.
